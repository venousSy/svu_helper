# UX Todo List

## Confirmation Summaries
Before finalizing a project submission, send a single message summarizing all inputs (Subject, Tutor, Date, Notes) with a [Confirm ✅] or [Edit ✏️] button. This reduces anxiety about making a mistake.

## SVU Terminology
Ensure the phrasing perfectly matches what students see on their SVU ISS and Moodle.
Instead of generic "Project" (مشروع), use the exact terms based on the context: الوظيفة (Assignment), حلقة البحث (Research Seminar), or مشروع التخرج (Graduation Project).

## File Naming Conventions & Formats
SVU often requires strict file naming (e.g., StudentID_Name_Subject.pdf). During the file upload step (ask_details), add a reminder: "تذكر: جامعة SVU ومسمى برقمك الجامعي". Have the bot automatically rename their uploaded file to match this standard before passing it to the admin.
